📖 玩法说明

1. 网格规则：黑色格子代表二进制 1，白色格子代表 0

2. 运算规则：

  • 每一行是一个二进制数，从上到下依次是行 0 ~ 行 n-1
  • 每一列也是一个二进制数，从左到右依次是列 0 ~ 列 n-1
  • 最终答案 = (所有行的 AND) OR (所有列的 XOR)

3. 目标：计算出最终结果，输入数字并验证

4. 提示：可以多次尝试，历史记录会保留你的猜测和结果
